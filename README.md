## ğŸ” **Troubleshooting**

### **Theme not loading?**
1. Check browser console for errors (F12)
2. Make sure URLs are correct (should be `raw.githubusercontent.com`)
3. Clear browser cache (Ctrl+Shift+Delete)
4. Make sure files are in the `main` branch

### **Files returning 404?**
- Wait 1-2 minutes after pushing to GitHub
- Verify repository is **Public**
- Check branch name (might be `master` instead of `main`)

### **Theme loads but looks broken?**
- Ensure all 3 files are loaded (check Network tab in DevTools)
- Verify file names match exactly in `Pinky.yml`
- Check that CSS/JS aren't blocked by browser extensions
